# Lexical-Complexity-Prediction
Prediction of Lexical Complexity using a sentence and a word in that sentence <br>

## Algorithms Implemented
* Bidirectional Encoder Representations from Transformers (BERT)

## Libraries Used
* Pandas
* CSV
* Numpy
* Transformers
* PyTorch
* Matplotlib
